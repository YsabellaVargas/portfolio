NU-IN Fashion (front-end development)

school opdracht voor Hogeschool van Amsterdam

Dit was een individuele opdracht vanuit het vak front-end development aan de HvA. Er moest een website worden gemaakt met netjes geschreven HTML & CSS code.


HET IDEE

Voor het vak front-end development wil ik de website van van NU-IN Fashion “namaken” in nette HML en CSS code. Ik houd de huisstijl aan en zal een aantal punten aan de lay-out veranderen omdat ik mij wil richten op het responsive toegankelijk maken van de website.


DOEL

Leren hoe ik zo netjes mogelijk een responsive website kan opleveren die er goed uit ziet en duidelijk geschreven code heeft. De code moet voorzien zijn van “comments” voor de overdraagbaarheid.

KLANT

Docent front-end development HvA

MIJN ROL

Ontwerper, Front-end developer
DE OPDRACHT

Twee pagina’s uitwerken van een (bestaande) website met nette HTML, CSS en Javascript code.

BREAKDOWN SCHETS

Voordat ik ben begonnen met het coderen van de website heb ik deze eerst “opgebroken” in stukken zodat ik voor mezelf een overzicht had hoe de website in elkaar zit.

HTML

Op onderstaande afbeelding is een schermafbeelding te zien van een stukje van mijn HTML code. Ik heb er op gelet waar ik sections en articles moest plaatsen om het zo overzichtelijk en netjes mogelijk te houden.

CSS

In de CSS heb ik met selectoren gewerkt omdat ik op deze manier leerde zo min mogelijk classes te gebruiken.

JAVASCRIPT

Om interactie in mijn website te verwerken heb ik een automatische slider aangemaakt die begint bij het aankomen op de home pagina.

Mijn volledige proces is ook te zien op mijn GitHub pagina:

Link naar GitHub pagina

CONCLUSIE

Met deze opdracht heb ik geleerd hoe ik een website responsive kan maken door met media queries te werken, hoe ik CSS selectoren gebruik en hoe flexbox werkt. Ik begrijp in welke situaties ik de juiste code moet gebruiken. Het was soms wel moeilijk omdat de website op beide devices goed moest werken, maar ik heb uiteindelijk een goed werkende website opgeleverd met interacties.
